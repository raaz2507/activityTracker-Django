About Activity Tracker

Activity Tracker App एक personal productivity और learning logger है। इसमें user अपनी daily activities जैसे पढ़ाई, खेल, फिल्म, गेम, किताब पढ़ना आदि track कर सकता है। App activity का source, reason/trigger, start time, end time और date save करता है। Main features में activity जोड़ना, calendar या list view में देखना, charts/graphs से visualize करना और analytics शामिल हैं। Future में CSV/Excel export।
